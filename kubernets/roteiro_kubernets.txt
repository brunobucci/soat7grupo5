Roteiro para o teste:

1- Criar as configMaps da API e do MySQL
kubectl apply -f api-configmap.yaml
kubectl apply -f mysql-configmap.yaml

2- Criar os Services para a API e o MySQL 
kubectl apply -f api-svc.yaml
kubectl apply -f mysql-svc.yaml

3- Criar o POD do MySQL
kubectl apply -f mysql-deployment.yaml

6- Criar o POD da API
kubectl apply -f api-deployment.yaml

11- Criar o component para as métricas - HPA
kubectl apply -f kube-metrics.yaml

12- Criar o HPA da APP
kubectl apply -f api-hpa.yaml

Comandos par testar:
#Checar o que está no ar
kubectl get pod,deployment,configmap,svc -o wide

#Keep Watch dos POD's
kubectl get pod --watch

#Keep Watch do HPA
kubectl get hpa --watch
