### Forma de execução das apis para execução da 2º fase

Passo 1 - Criação de um produto
  POST http://{servidor}:{porta}/produtos
  Exemplo de JSON
  {
    "idProduto": 0,
    "nome": "Nome do produto",
    "descricao": "Descrição do produto.",
    "preco": 10.0,
    "duracaoPreparo": 15,
    "idTipo": 2,
    "flAtivo": 1
  }

Passo 2 – Checkout Pedido
  POST http://{servidor}:{porta}/pedidos/checkout
  Exemplo de JSON
  {
    "pedido": {
      "idPedido": 0,
      "preco": 0,
      "idStatusPedido": 0,
      "idCliente": 1,
      "idTipoPagamento": 1,
      "duracaoTotalPreparo": 0,
      "dataCriacao": "2024-07-19T19:12:33.040Z",
      "idStatusPagamento" : 0
    },   
    "pedidoProdutos": [
      {
        "idPedidoProduto": 0,
        "idPedido": 0,
        "idProduto": 6,
        "precoProduto": 11,
        "nomeProduto": "Cheese Burger",
        "duracaoPreparo": 7
      },

  {
        "idPedidoProduto": 0,
        "idPedido": 0,
        "idProduto": 102,
        "precoProduto": 10,
        "nomeProduto": "Fritas",
        "duracaoPreparo": 5
      },

  {
        "idPedidoProduto": 0,
        "idPedido": 0,
        "idProduto": 3,
        "precoProduto": 12,
        "nomeProduto": "Suco de laranja",
        "duracaoPreparo": 7
      }
    ]
  }

Passo 3 – Consultar status de pagamento do pedido
  GET http://{servidor}:{porta}/pedidos/status-pagamento/{idPedido}
  Exemplo de consumo: http://localhost:8081/pedidos/status-pagamento/554 

Passo 4 – Webhook para aprovar ou reprovar pagamento
  POST http://{servidor}:{porta}/pedidos/checkout
  Exemplo de JSON
  {
    "idPedido": 554,
    "pagamentoAprovado": true
  }

Passo 5 – Listagem ordenada
  GET http://{servidor}:{porta}/pedidos/busca-ordenada
  Exemplo de consumo: http://localhost:8081/pedidos/busca-ordenada

Passo 6 – Atualização de status do pedido
  PUT http://{servidor}:{porta}/pedidos/{idPedido}/status/{idStatus}
  Exemplo de consumo: http://localhost:8081/pedidos/1/status/1
